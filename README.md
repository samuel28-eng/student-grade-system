Student Grade Management System

 Project Overview
This project is a simple Python-based system that collects a student's name and score, calculates the grade, and displays the result.

## Software Development Life Cycle (SDLC)

### 1. Requirement Analysis
The system is designed to accept student details, compute grades, and display results.

### 2. System Design
The system uses a simple console-based design with conditional statements for grading.

### 3. Implementation
The project is implemented using Python.

### 4. Testing
Different score ranges were tested to ensure correct grade assignment.

### 5. Deployment
The project is deployed by uploading it to GitHub.

### 6. Maintenance
Future updates may include storing multiple student records.

## Technology Used
- Python
